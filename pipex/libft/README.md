## **Libft**
- 함수 구현해서 나만의 라이브러리 만들기
- Technical Considerations
  - 전역변수 사용 금지
  - 함수 구현을 위해 서브 함수를 만들어야 한다면, `static` 키워드 사용
  - `ar` 명령어를 사용해 librairy 생성
